回忆线性代数，可能有唯一解，无穷多解，无解。

即求非齐次线性方程组的解，列增广矩阵即可。所谓高斯消元也和线性代数中矩阵的初等行变换异曲同工。即反复做初等行变换成行阶梯型矩阵，r(a)=r(a∣b)+1r(a) = r(a|b)+1r(a)=r(a∣b)+1则无解，r(a)=r(a∣b)<nr(a) = r(a|b) < nr(a)=r(a∣b)<n则有无穷多解，r(a)=r(a∣b)=nr(a) = r(a|b) = nr(a)=r(a∣b)=n则有唯一解。线性代数基础知识在这里不再赘述。

当然了，给计算机的算法不能像在演算时做的初等行变化那么随意，或者是为了好算凑整数，我们需要制定一个步骤，让算法反复执行。
转化成高斯消元的算法步骤如下：

  1. 枚举每一列，找到该列绝对值最大的一行
  2. 将该行交换到最上面
  3. 将该行第一个数变为1
  4. 消除该列其下所有行为0

我们当然可以按照组合数的公式来直接求解，即
a×a−1×...(a−b+1)1×2××...×b=a!b!(a−b)!\frac{a\times a-1\times ... (a-b+1)}{1\times 2\times \times ... \times b} = \frac{a!}{b!(a-b)!}1×2××...×ba×a−1×...(a−b+1)​=b!(a−b)!a!​

由于进行了阶乘运算，在a,b范围之内会超时，因此不能直接暴力运算。实际上，我们可以通过一个递推式来运算：

满足条件的01序列

给定nnn个 0 和nnn个 1，它们将按照某种顺序排成长度为 2n2n2n的序列，求它们能排列成的所有序列中，能够满足任意前缀序列中 0 的个数都不少于 1 的个数的序列有多少个。输出答案对109+710^9+7109+7取模

令人震惊的是本题中用到了卡特兰数，这个在数据结构中反复出现了n多次的关键定理。

假设一个n×nn\times nn×n的棋盘，从原点出发走到右上角，0为向x轴正方向前进一个单位，1为向y轴正方向前进一个单位，那么从原点到终点对应的每条路径就是一个对应的序列，那么为满足题设0的个数不少于1的个数，就要求路径上任何一个位置都要满足x>=y这个条件，故在y = x + 1这条线的下方。所以合法的路径就相当于所有路径减去不合法的路径数目，而所有路径数显然等于C2nnC_{2n}^{n}C2nn​，不合法的路径数，考虑将这些不合法的路径数从讲过y = x + 1的点之后沿y = x + 1轴对称，那么所有的不合法路径均可以转化为一条从(0,0)到(n-1,n+1)的路径，这些不合法路径总数为C2nn−1C_{2n}^{n-1}C2nn−1​，最终的合法路径将二者相减，展开，合法的路径数目恰好就是卡特兰数：C2nnn+1\frac{C_{2n}^{n}}{n+1}n+1C2nn​​。那么我们的算法如下：
