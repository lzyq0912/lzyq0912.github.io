欧拉函数

欧拉函数的定义：

1∼N1 \sim N1∼N中与NNN互质的数的个数被称为欧拉函数，记为ϕ(N)\phi(N)ϕ(N)。

互质的定义是 gcd(a,b)=1gcd(a,b) = 1gcd(a,b)=1

若在算数基本定理中，N=p1a1p2a2...pmamN = p_1^{a_1}p_2^{a_2}...p_m^{a_m}N=p1a1​​p2a2​​...pmam​​，则有：
ϕ(N)=N×p1−1p1×p2−1p2×...×pm−1pm\phi(N) = N \times \frac{p_1-1}{p_1}\times \frac{p_2-1}{p_2} \times ...\times\frac{p_m-1}{p_m}ϕ(N)=N×p1​p1​−1​×p2​p2​−1​×...×pm​pm​−1​

证明

这个公式怎么来的呢？，实际上用到了容斥原理；

  1. 显然我们分解质因数之后可以得到NNN的质因子为p1,p2,...,pmp_1,p_2,...,p_mp1​,p2​,...,pm​；那么我们从1∼N1 \sim N1∼N中去掉p1,p2,...,pmp_1,p_2,...,p_mp1​,p2​,...,pm​的所有倍数。
  2. 第一步中去掉的数可能是重复的，例如某个数可能同时是p1,p2p_1,p_2p1​,p2​的倍数，因此之后的操作是加上所有pi×pjp_i \times p_jpi​×pj​的倍数，这些数在第一步操作中被减了两次。
  3. 考虑p1,p2,p3p_1,p_2,p_3p1​,p2​,p3​的共同倍数呢？他们首先在第一步中被减了三次，又在第二步中被加了三次，然而我们的目的是将他们减去一次，因此我们需要再减去所有pi,pj,pkp_i,p_j,p_kpi​,pj​,pk​的倍数。
  4. 观察到这里，我们可以发现最后的结果，与N互质的数的个数为：
    N−N/p1−N/p2−...−N/pk+N/p1p2+N/p1p3+...−N/p1p2p3−N/p1p2p4−...+N/p1p2p3p4N - N/p_1-N/p_2-...-N/p_k+N/p_1p_2 + N/p_1p_3+...-N/p_1p_2p_3-N/p_1p_2p_4-...+N/p_1p_2p_3p_4N−N/p1​−N/p2​−...−N/pk​+N/p1​p2​+N/p1​p3​+...−N/p1​p2​p3​−N/p1​p2​p4​−...+N/p1​p2​p3​p4​
    而上面这个超极长的式子利用容斥原理组合，恰好就是欧拉函数

欧拉函数的应用

用于欧拉定理：若aaa与nnn互质，则aϕ(n)≡1(modn)a^{\phi(n)}\equiv 1\pmod naϕ(n)≡1(modn)，其中ϕ(n)\phi(n)ϕ(n)为欧拉函数。

那么欧拉定理的质数特例即为：ap−1≡1(modp)a^{p-1}\equiv 1\pmod pap−1≡1(modp)，这也被称为费马小定理。

欧拉定理的简易证明如下：

设集合 $A = { x_1(\text{mod }m), x_2(\text{mod }m),\dots, x_{\varphi{(m)}}(\text{mod }m) } $是小于 mmm 的所有与 mmm 互素的不同的数的集合(这里取模是为了与后面形式上同一，不取模肯定也是小于 mmm 的)。

然后再令 B=ax1(mod m),ax2(mod m),…,axφ(m)(mod m)B = { ax_1(\text{mod }m), ax_2(\text{mod }m),\dots,ax_{\varphi{(m)}}(\text{mod }m) }B=ax1​(mod m),ax2​(mod m),…,axφ(m)​(mod m)
因为都取了模，所以 BBB 中所有元素也是小于 mmm 的. 又因为 aaa 与 mmm 互素，每一个 xxx 都与 mmm 互素，由互素整数的性质三可以得到ax1ax_1ax1​, ax2ax_2ax2​,…,axmax_maxm​ 都与 mmm 互素, 即(axi,m)=1,i=1,2,…,m(ax_i, m) = 1, i = 1, 2, \dots, m(axi​,m)=1,i=1,2,…,m，
由裴蜀定理我们可以得到
(集合B中每一个元素,m)=1(\text{集合}B\text{中每一个元素}, m) = 1(集合B中每一个元素,m)=1
并且取完模后 BBB 中的元素已经都小于 mmm了. 我们还可以进一步说明，BBB 中所有的元素两两不同，若 BBB 中存在相同的元素，即假设存在 axi(mod m)=axj(mod m)  ⟺  axi≡axj(mod m),ax_i(\text{mod }m) = ax_j(\text{mod }m) \iff ax_i \equiv ax_j (\text{mod }m),axi​(mod m)=axj​(mod m)⟺axi​≡axj​(mod m), 那么因为 aaa 与 mmm 是互素的，所以 aaa 可以直接约掉，那么就会有i≠j,i\ne j,i=j, 有 xi(mod m)=xj(mod m)  ⟺  xi≡xj(mod m)x_i(\text{mod }m) = x_j(\text{mod }m)\iff x_i \equiv x_j (\text{mod }m)xi​(mod m)=xj​(mod m)⟺xi​≡xj​(mod m) 这就与A集合中元素互不相同这一前提出现了矛盾. 所以 BBB 中元素也互不相同且和 mmm 互素，并且 AAA 集合 与 BBB 集合元素的个数是相等的. 我们已经定义AAA 是所有小于 mmm 且与 mmm 互素的整数个数，那么BBB 中的这些元素肯定也是 AAA 中这些元素，这不过是顺序换了一下，专业术语叫做集合 AAA 元素的一个置换.
那么把集合 AAA 内所有元素相乘与 BBB 内所有元素相乘是相等的，可以得到以下式子
x1x2…xφ(m)≡ax1ax2…axφ(m)(mod m) x_1x_2\dots x_{\varphi({m})} \equiv ax_1ax_2\dots ax_{\varphi({m})}(\text{mod }m) x1​x2​…xφ(m)​≡ax1​ax2​…axφ(m)​(mod m)
x1x2…xφ(m)≡aφ(m)x1x2…xφ(m)(mod m) x_1x_2\dots x_{\varphi({m})} \equiv a^{\varphi(m)}x_1x_2\dots x_{\varphi({m})}(\text{mod }m) x1​x2​…xφ(m)​≡aφ(m)x1​x2​…xφ(m)​(mod m)
所以
aφ(m)≡1(mod m) a^{\varphi(m)}\equiv 1(\text{mod }m) aφ(m)≡1(mod m)
以上就证明了欧拉定理。

由ak=a21xa22x...a2tx=a21x+22x+...+2txa^k = a^{2^x_1}a^{2^x_2}...a^{2^x_t} = a^{2^x_1+2^x_2+...+2^x_t}ak=a21x​a22x​...a2tx​=a21x​+22x​+...+2tx​
因此我们要计算ak mod pa^k \:mod \:pakmodp只需依次预处理a20 mod p;a21 mod p;...;a2log⁡k mod pa^{2^0}\:mod\:p;a^{2^1}\:mod\:p;...;a^{2^{\log k}}\:mod\:pa20modp;a21modp;...;a2logkmodp
之后再组合起来即可，因此我们只需要将kkk拆解成若干个二次幂的和。这里直接利用二进制优化，将kkk写成二进制的数再展开即可。因此时间复杂度总体为O(log⁡k)O(\log k)O(logk)
